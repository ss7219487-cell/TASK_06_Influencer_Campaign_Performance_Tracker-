TASK_06_Influencer_Campaign_Performance_Tracker

## Problem Statement
Brands invest heavily in influencer marketing but often struggle to identify which campaigns, influencers, and platforms deliver the best return on investment (ROI). This project develops an interactive Power BI dashboard to analyze campaign performance and support data-driven marketing decisions.

## Dataset Description
- Dataset: Final_Influencer_Campaign_Dataset.csv
- Domain: Influencer Marketing Analytics
- Records: 1,048,575

## KPI Definitions
- Total Campaigns
- Total Revenue
- Total Spend
- Total Reach
- Total Conversions
- Average ROI
- Average Engagement Rate
- Cost Per Engagement

## Dashboard Pages
### Page 1 – Dashboard Overview
- KPI Cards

- Platform Analysis
- Audience Analysis

### Page 2 – Influencer Performance Analysis
- Average ROI
- Average Engagement Rate
- Average Followers
- Average Conversions
- Top Influencers
- ROI by Influencer Size

### Page 3 – Influencer Segmentation
- Influencer Size Distribution
- ROI Category Distribution
- Engagement Category Distribution
- Cost Efficient Campaign Distribution

## Key Findings
- High ROI influencers produce better campaign results.
- Platform performance varies.
- Audience groups respond differently.
- Cost-efficient campaigns improve ROI.

## Recommendations
- Focus on high ROI influencers.
- Optimize campaign spending.
- Improve engagement strategy.
- Monitor KPIs regularly.
